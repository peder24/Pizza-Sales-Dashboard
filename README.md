![alt_text](https://res.cloudinary.com/dk2tex4to/image/upload/v1783696636/Screenshot_2026-07-10_221645_urt1oj.png)

## Business Questions

This dashboard is designed to answer the following questions:

- How much revenue was generated?
- How many orders and pizzas were sold?
- Which pizza category performed the best?
- Which pizza size contributed the most sales?
- Which days and months recorded the highest number of orders?

## KPIs

- Total Revenue
- Average Order Value
- Total Pizzas Sold
- Total Orders
- Average Pizzas per Order

## Key Insights

- Total revenue reached **817.86K** from **21,350 orders**.
- Customers purchased an average of **2.32 pizzas per order**.
- Friday recorded the highest number of orders, while Sunday had the lowest.
- July had the highest monthly order volume, whereas October had the lowest.
- The Classic category sold the highest number of pizzas.
- Large pizzas contributed the largest share of total sales.
